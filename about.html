Goals

In this class, I hope to become more comfortable using GitHub as a tool for collaboration and version control. I want to learn how to push and pull files, track changes in my projects, and organize my work in a way that feels professional. I also hope to get a better understanding of how GitHub is used in real-world jobs so I can apply these skills outside of the classroom.With VS Code, my goal is to build confidence in writing and editing code. I want to learn how to structure my projects correctly, use extensions to make coding easier, and troubleshoot errors on my own.

I am especially passionate about digital media and communication.
